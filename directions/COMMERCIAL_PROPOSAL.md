# ЧАСТЬ 1. ЧТО ВЫ ПОЛУЧАЕТЕ

## Полностью автоматическая система записи

| Без системы | С системой |
|------------|-----------|
| Клиент пишет в Instagram | Клиент открывает бота и записывается сам |
| Менеджер уточняет время вручную | Бот показывает реальное расписание и фиксирует слот |
| Менеджер создаёт Zoom и отправляет ссылку | Zoom создаётся автоматически за секунды |
| Менеджер напоминает о занятии | 2 напоминания автоматически — за 24ч и за 5ч |
| Клиент забыл — узнали когда не пришёл | Менеджер получает алёрт за 4ч если не подтвердил |
| Данные в голове менеджера или в разных местах | Всё в одной локальной базе — имя, телефон, email, статус |

---

### Что входит в систему:

**1. Telegram-бот для клиентов**
Профессиональный бот. Клиент проходит запись за 2–3 минуты прямо в мессенджере.

**2. Google Calendar интеграция**
Бот показывает только реальные свободные слоты из вашего расписания.

**3. Автоматический Zoom**
После бронирования — Zoom-ссылка создаётся и отправляется клиенту без участия менеджера.

**4. Email-цепочка (Resend)**
Два письма: через 30 минут после заявки и через 24 часа. Если не записался — статус "нужен звонок".

**5. AI-ассистент**
Отвечает на вопросы о школе 24/7. Голосом или текстом. Мягко ведёт к записи на урок.

**6. Локальная CRM (ваши данные — только у вас)**
Вся база клиентов хранится на вашем сервере в SQLite. Никаких Google, никаких третьих сторон. Доступ через admin-бот.

**7. Telegram Admin-панель**
Отдельный бот для вас — статистика, расписание на день, рассылки, посещаемость, карточки клиентов, редактирование текстов и AI-промта.

**8. Интеграция с сайтом (Tilda)**
Заявка с сайта → автоматически в CRM → бот начинает работу с клиентом.

# ЧАСТЬ 5. НАДЁЖНОСТЬ — ЧЕСТНЫЙ ОТВЕТ

## Что работает стабильно

✅ **Telegram** — 99.9% uptime, автопереподключение при сбоях

✅ **SQLite (локальная база)** — данные на вашем сервере. Никаких внешних зависимостей. Работает без интернета. Мгновенные чтение/запись.

✅ **Google Calendar** — только для слотов. Если Google упадёт — бронирование временно недоступно, всё остальное работает.

✅ **Zoom** — токен обновляется автоматически каждые 50 минут. Если Zoom упал — бронирование фиксируется, ссылка отправляется отдельно.

✅ **OpenAI** — если недоступен, AI-ассистент отключается, всё остальное продолжает работать.

✅ **Планировщик напоминаний** — задачи сохраняются в файл. При перезапуске сервера — восстанавливаются автоматически.

## Почему локальная база (SQLite) лучше Google Sheets для CRM

| | Google Sheets | SQLite (текущее решение) |
|--|---------------|--------------------------|
| Скорость | 200–500мс на запрос | <1мс |
| Зависимость | API Google, OAuth токен | Ноль внешних зависимостей |
| Данные | На серверах Google (третья сторона) | Только на вашем сервере |
| GDPR (Эстония) | Требует DPA с Google | Полное соответствие автоматически |
| Лимиты | 100 запросов/сек | Без лимитов |
| Стоимость | €6/мес (Google Workspace) | €0 |
